1. Создать папку
2. Поместить в нее:
  1) Переменованный в statement.xls файл выгрузки
	а) Открыть и переименовать лист в необходимую дату из отчета(Факт) и сохранить. Файл может быть открыт во время исполнения пункта 6.
  2) Переменованный в report.xlsx файл отчета(Факт). Файл должен быть закрыт во время исполнения пункта 6.
  3) jar`ник с програмулей
3. Скопировать адрес папки, в которую ты поместила файлики из пункта 2.
4. Нажать комбинацию "win+r", ввести cmd и нажать "enter"
5. В коммандной строке(cmd, который ты открыла в пункте 4.) набрать cd, пробел и вставить адрес папки из пункта 3.
6. В коммандной строке набрать java -jar statementToReport-1.0.jar
